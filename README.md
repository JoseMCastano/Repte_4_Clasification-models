# Repte_4_Clasification-models

## Modelo predictivo de tumores de mama

El objetivo del ejercicio es el desarrollo de un modelo predictivo de clasificaci√≥n, que permita a partir de unas variables asociadas a un tumor de mama, predecir si se trata de un tumor benigno o maligno.
